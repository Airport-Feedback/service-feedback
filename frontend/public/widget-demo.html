Embeddable Feedback Widget

This widget allows you to collect visitor feedback on any website. It's completely self-contained and won't interfere with your existing site.

Basic Implementation

Add this script tag to your website:

<script>
window.FeedbackWidgetConfig = {
    companyName: 'Your Company',
    position: 'bottom-right',
    primaryColor: '#365fb8',
    baseUrl: 'https://your-domain.com'
};
</script>
<script src="https://your-domain.com/embed.js"></script>

Configuration Options

Position Options:

bottom-right (default)
Bottom right corner
bottom-left
Bottom left corner
top-right
Top right corner
top-left
Top left corner

Color Customization:

Use any valid CSS color value:

  • primaryColor: '#ff6b6b' - Hex color
  • primaryColor: 'rgb(255, 107, 107)' - RGB color
  • primaryColor: 'hsl(0, 100%, 71%)' - HSL color

Alternative: Direct iFrame Embed

If you prefer not to use JavaScript, you can embed the widget directly as an iframe:

<iframe 
    src="https://your-domain.com/#/widget?company=Your%20Company&position=bottom-right&color=%23365fb8"
    style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 999999; pointer-events: auto; background: transparent;"
></iframe>

Features

  • ✅ 5-point emoji rating system
  • ✅ Optional text feedback
  • ✅ Optional email collection
  • ✅ Offline-first data storage
  • ✅ Automatic data synchronization
  • ✅ Customizable appearance
  • ✅ Mobile responsive
  • ✅ No dependencies on your site
